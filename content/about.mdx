## 关于西江月

**西江月**是一个技术博客，记录云原生、容器部署、全栈开发和自动化运维等方面的实践笔记。

### 🧱 站点技术

本站基于 Next.js 16 App Router + Tailwind CSS v4 + MDX 构建，部署于 Vercel。视觉设计移植自 [QingHome](https://github.com/yuanjia1314/QingHome)。

### 🔗 技术来源

博客的搭建技术和基础设施参考自 [雨天狂奔](https://github.com/yuanjia1314) 的开源实践：

- [nav-site](https://github.com/yuanjia1314/nav-site) — API 导航站，Next.js + Supabase + Redis
- [QingHome](https://github.com/yuanjia1314/QingHome) — 个人主页设计系统
- [domain-check](https://github.com/yuanjia1314/domain-check) — 域名监控，Cloudflare Workers
- [blog](https://github.com/yuanjia1314/blog) — 本站源码

### 📋 技术栈速览

前端：React / Next.js / TypeScript / Tailwind CSS
后端：Node.js / Go / Python / PostgreSQL
云服务：Cloudflare Workers / Supabase / Docker
工具链：Git / GitHub Actions / pnpm
